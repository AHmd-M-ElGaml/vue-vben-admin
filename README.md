## Quick Start

1. Clone the repository:

```bash
git clone https://github.com/AHmd-M-ElGaml/vue-vben-admin.git
```

2. Install dependencies:

```bash
cd vue-vben-admin
npm i -g corepack
pnpm install
```

3. Start development server:

```bash
pnpm dev
```

4. Build for production:

```bash
pnpm build
```
